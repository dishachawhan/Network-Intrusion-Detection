# Network Intrusion Detection System

## Overview
This project uses Machine Learning to detect malicious network traffic and identify potential cyber attacks. It is trained on the NSL-KDD dataset.

## Features
- Data preprocessing and encoding
- Random Forest classifier
- Binary classification (Normal vs Attack)
- Confusion matrix visualization
- Feature importance analysis
- Real-time prediction system
- Interactive Streamlit dashboard

## Dataset
NSL-KDD dataset is used for network intrusion detection research.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit

## Model Performance
- Accuracy: ~99.86%  
- Confusion Matrix and Classification Report generated
